# Decimal to Binary Converter

C#で作成した、10進数を2進数に変換するコンソールアプリです。

## 概要
0以上65535以下の整数値を入力すると、対応する2進数を表示します。  
`end` と入力するとアプリを終了します。

## 特徴
- 入力値の検証（整数判定・範囲チェック）
- 2進数変換を自作ロジックで実装
- `Convert.ToString` 等の組み込み関数は未使用

## 実行方法
1. Visual Studioでプロジェクトを開く
2. ビルドして実行
3. 10進数を入力
4. `end` を入力すると終了

## 使用技術
- C#
- .NET（Console Application）

## 学習目的
2進数の仕組みと、  
C#での制御構文・配列・メソッド分割の理解を目的として作成しました。
